> *A curated collection of programs, practicals, and projects spanning Java, C, Web Development, Spring Framework, and more — built as part of my learning journey.*

This repo is my personal code playground — from core programming fundamentals to real-world mini-projects. Everything you'll find here reflects hands-on practice, academic assignments, and self-driven exploration.

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/Chaitanya-Rasal/Programs.git

# 2. Navigate into it
cd Programs

# 3. Pick any folder and explore!
cd basics   # or oops, jdbc, etc.
```

> **For Java files:** Compile with `javac FileName.java` and run with `java FileName`  
> **For C files:** Compile with `gcc filename.c -o output` and run with `./output`
